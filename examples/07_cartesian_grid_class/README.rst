CartesianMesh
=============

Examples demonstrating the :class:`~pyvisual.core.mesh3d.CartesianMesh` class
— the Cartesian-frame counterpart to :class:`~pyvisual.core.mesh3d.SphericalMesh`.
Topics include construction from NumPy arrays, coordinate-frame conversion from
a :class:`~pyvisual.core.mesh3d.SphericalMesh`, and direct rendering via
:class:`~pyvisual.core.plot3d.Plot3d`.
